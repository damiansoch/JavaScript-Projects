Counting to 10

Player needs to choose a whole number (0-99). Computer will randomly get the second number. In order to win, these numbers need to be equal. There are 100 chances (rounds). Max number of points is 100, every round player is losing 1 point. So for example: if the numbers are matched in round 50, the player will get 50 points. If they are matched on round 99, the player will get 1 point. If the numbers don't match at all during 100 rounds - 0 points… Good Luck… :D
The program first will try to parse the integer from the input and than the program checks if:
  1. input isn't empty
  2. input is in 0-99 range
  3. in case of mixed numbers (5.8 or 5 1/2), the whole number will be used and the program will get rid of the fraction piece.
